Whimsical Wardrobe: OSASCOMP Explorer

A visual, interactive tool designed for ESL False Beginners to practice the complex rules of English adjective order (OSASCOMP). This project turns a rigid grammar lesson into a creative "Fashion Runway" activity.

🌟 Overview

Teaching adjective order (Opinion, Size, Age, Shape, Color, Origin, Material, Purpose) can be overwhelming. This tool provides students with high-quality, AI-generated "Whimsical Wardrobe" items and a randomized word bank. Students select adjectives to "dress" the noun and plan their sentences before submitting them for class review.

🚀 Setup Instructions

This repository is designed to be hosted on GitHub Pages as a static site.

Upload Files: Ensure Whimsical Wardrobe Final.html and all 10 PNG images are in the same root folder.

Naming Convention: Do not rename the image files, as the HTML code specifically looks for the filenames provided (e.g., Whimsical Wardrobe - The Steampunk Boots.png).

Enable GitHub Pages:

Go to your Repository Settings.

Click on Pages in the left sidebar.

Select Deploy from a branch and choose main (or master).

Your site will be live at https://your-username.github.io/your-repo-name/Whimsical%20Wardrobe%20Final.html.

🎓 How to Play

1. The Presentation

Project the GitHub Pages link onto your classroom screen. Move through the 10 rounds together as a class.

2. The Task

For each item, students are instructed to:

Look at the whimsical image.

Select 4 or 5 adjectives from the randomized word bank that they feel best describe the item.

Arrange those adjectives in the correct OSASCOMP order.

3. Submission

Since this is a static page for maximum compatibility, students submit their final noun phrases via a Google Form (linked separately in your Canvas or LMS).

🧠 Pedagogical Focus: OSASCOMP

The word banks are carefully curated for beginners:

Opinion: Nice, Cool, Pretty, Good

Size: Small, Big, Long, Tall

Age: Old, New

Shape: Round, Square, High, Sharp

Color: Brown, Blue, Purple, Green, Red, Gold, Clear

Origin: English, Space, Nordic, City, Garden, Forest, School, Magic, Star

Material: Leather, Silk, Ice, Metal, Cloth, Feather, Glass

Purpose: Flying, Fighting, Working, Party, Royal, Walking, Testing, Dancing, Timing

🛠️ Technical Details

Framework: Built with React (via CDN) and Tailwind CSS.

No Backend: Requires no database or external API keys.

Responsive: Works on desktops, tablets, and mobile phones.

Created as an Open Educational Resource (OER) for ESL 402/410.
